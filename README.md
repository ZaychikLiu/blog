# Hexo Icarus Blog Template

This is a Hexo + Icarus blog template modeled after the structure of <https://ppwwyyxx.com/blog/>.

It includes:

- Hexo 7.x
- Icarus theme
- `/blog/` GitHub Pages subpath configuration
- Archives, tags, about page, RSS
- profile/sidebar widgets
- sample posts
- optional advanced features: comments, analytics, share buttons, reading progress, table of contents, popular posts, sitemap, robots.txt
- GitHub Actions deployment
- Chinese content guide in `docs/CONTENT_GUIDE.md`

## Quick Start

```bash
npm install
npm run dev
```

Then open:

```text
http://localhost:4000/blog/
```

## Build

```bash
npm run build
```

The static site will be generated into `public/`.
